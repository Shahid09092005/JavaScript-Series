Cheat Sheet for important methods


| Method      | Returns        | Purpose                    |
| ----------- | -------------- | -------------------------- |
| **forEach** | nothing        | Looping only               |
| **map**     | new array      | Transform values           |
| **filter**  | new array      | Keep only matching values  |
| **reduce**  | single value   | Calculate (sum, max, etc.) |
| **find**    | single element | First match                |
| **some**    | boolean        | At least one matches       |
| **every**   | boolean        | All must match             |
